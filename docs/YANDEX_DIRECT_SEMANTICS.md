# MADCORE Gena Yandex Direct Semantics

## Текущий статус

Семантика для `MADCORE Gena` должна собираться отдельно от исходного сайта. На этой точке готова только схема сегментов и инфраструктурная база.

## Отдельные направления объявлений

1. Сайт
2. Переход в Telegram
3. Переход в WhatsApp
4. Переход в MaX
5. Телеграм-каналы

## Что обязательно сохранить

- отдельные `cmp`;
- отдельные UTM;
- отдельные цели Метрики;
- отдельные отчеты качества лидов.

## Следующий шаг

После создания отдельных кампаний занести сюда:

- ID кампаний;
- структуру групп;
- тексты объявлений;
- минус-слова;
- правила модерации и запуска без бюджета.
