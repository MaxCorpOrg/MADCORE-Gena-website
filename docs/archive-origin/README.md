# Архив исходного MADCORE

Этот каталог хранит материалы, унаследованные из исходного `MADCORE-website` при создании `MADCORE Gena`.

Их нельзя считать актуальной документацией нового проекта без отдельной перепроверки:

- в них могут встречаться старый домен;
- старые номера и CTA;
- старые рекламные `cmp`;
- старые live ID Метрики, Matomo и production-заметки.

Для текущей работы по `MADCORE Gena` использовать только документы из корня `docs/` и актуальные checkpoint-файлы.
