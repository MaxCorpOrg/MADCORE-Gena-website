# Минус-слова Яндекс.Директа для MADCORE

Обновлено: `2026-05-17`

## Назначение

Этот документ фиксирует точный стартовый список минус-слов и минус-фраз для трех обычных кампаний `MADCORE`:

- `709948992` — сайт
- `709948993` — переход в Telegram
- `709948994` — переход в WhatsApp

Документ нужен, чтобы:

- убрать явный информационный и мусорный спрос;
- не смешивать продуктовую рекламу с обучающими, медицинскими и сервисно-чужими запросами;
- подготовить безопасную первую волну перед модерацией;
- не срезать слишком широко живой коммерческий спрос.

Готовые import-ready файлы:

- `docs/direct-import/yandex-minus-words-common.txt`
- `docs/direct-import/yandex-minus-words-review.txt`

## На чем основан список

1. Текущий сайт и продуктовые тексты:
   - `src/config/site.ts`
   - live `https://madcore-kavkaz.ru`
2. Уже загруженная семантика:
   - `docs/YANDEX_DIRECT_SEMANTICS.md`
3. Живые подсказки Яндекса по запросам:
   - `жиросжигание`
   - `снижение веса`
   - `сушка тела`
   - `коррекция фигуры`
   - `проблемные зоны`
   - `сухая мышечная масса`
   - `восстановление после тренировок`
   - `madcore 2.0`
4. Вывод из подсказок:
   - по широким темам много информационного спроса;
   - по темам `снижение веса`, `сушка`, `убрать живот`, `жиросжигание` Яндекс очень часто тянет:
     - упражнения
     - меню
     - отзывы
     - фото
     - медицинские причины
     - домашние сценарии
   - значит, коммерческую часть нужно защищать точными минус-фразами, а не надеяться только на ключи.

## Главный вывод по исследованию

- Общие темы вроде `снижение веса`, `сушка тела`, `жиросжигание`, `убрать живот` тянут очень много некоммерческого спроса.
- Самый частый мусор:
  - `в домашних условиях`
  - `меню`
  - `упражнения`
  - `как ...`
  - `что это`
  - `причины`
  - `отзывы`
  - `фото`
  - медицинские состояния и частные диагнозы
- Поэтому для `MADCORE` нельзя ограничиваться только ключевыми фразами; базовый минус-словарь должен быть жестким, но не слишком широким.

## Что добавлять не сразу, а после ручной проверки

Эти слова часто дают слабый или исследовательский трафик, но могут встречаться и в позднем коммерческом спросе. Их не надо бездумно срезать первой же загрузкой.

```text
отзывы
реальные отзывы
отзывы и результаты
до и после
до и после фото
состав
инструкция
побочные
побочки
аналоги
сравнение
айрекоменд
отзовик
```

Решение по этому блоку:

- если задача — максимально чистый лидовый трафик на первой волне, эти слова можно добавить позже отдельным шагом;
- если задача — не терять горячих пользователей на стадии проверки доверия, сначала оставить их без минусов и наблюдать по качеству лидов.

## Что не надо минусовать широко

Эти слова звучат спорно, но для `MADCORE` их нельзя автоматически резать на уровне всех кампаний:

- `цена`
- `купить`
- `заказать`
- `наличие`
- `официальный`
- `консультация`
- `Telegram`
- `WhatsApp`
- `Нальчик`
- `КБР`
- `Северный Кавказ`
- `как заказать`

Причина:

- они усиливают коммерческое намерение;
- часть из них уже входит в рабочие загруженные фразы;
- широкий запрет сломает полезный лидовый спрос.

## Практическая схема применения

1. Сначала загрузить только базовый общий список.
2. Не добавлять сразу блок `отзывы / состав / инструкция / аналоги`.
3. После первых данных смотреть:
   - поисковые запросы
   - качество лидов
   - `scoreIn`
   - `scoreOut`
4. Только потом точечно усиливать минус-фразы по группам.
