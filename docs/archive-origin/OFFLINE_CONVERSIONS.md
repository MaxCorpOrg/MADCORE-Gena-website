# Офлайн-конверсии для Яндекса

## Зачем это нужно

Офлайн-конверсии нужны для передачи в Яндекс не любого действия на сайте, а только реально подтвержденного результата:

- качественного лида
- продажи

Именно этот слой должен использоваться для дальнейшей оптимизации рекламных кампаний.

## Основная связка идентификаторов

В проекте используется такая цепочка:

- `click_id` и `yclid` приходят во входной рекламный маршрут `/go`
- Метрика после загрузки страницы выдает `ClientID`
- заявка сохраняется в `leads` с `yclid` и `client_id`
- менеджер ставит статус `qualified` или `sale`
- после этого лид попадает в офлайн-выгрузку

Главный технический идентификатор для Метрики:

- `ClientID`

Дополнительный рекламный идентификатор:

- `yclid`

## Что попадает в выгрузку

Только лиды, у которых:

- есть `ClientID` или `yclid`
- статус `qualified` или `sale`
- либо указана сумма продажи

## Что не должно попадать

- простые клики по CTA
- `form_start`
- неподтвержденные заявки
- дубли
- плохие лиды

## Как выгрузить

```bash
DAYS=30 ./scripts/export-qualified-leads-for-yandex.sh
```

Если shell-переменные `POSTGRES_USER` и `POSTGRES_DB` не экспортированы вручную, сценарий сам берет их из `/opt/madcore-gena/.env`.

## Что получается на выходе

CSV-файл в серверной папке отчетов:

- `ClientID`
- `yclid`
- `Target`
- `DateTime`
- `Price`
- `Currency`

`DateTime` выгружается как Unix timestamp, что соответствует требованиям Яндекса для офлайн-конверсий.

## Как использовать дальше

1. Проверить CSV вручную.
2. Убедиться, что в нем нет неподтвержденных лидов.
3. Загрузить файл в Яндекс.Метрику как офлайн-конверсии.
4. Использовать `qualified_lead` как основную качественную цель.
5. Использовать `sale` как более поздний уровень оптимизации.

## Важное отличие от JavaScript-целей

На сайте обычными JavaScript-целями остаются:

- `telegram_click`
- `whatsapp_click`
- `call_click`
- `consultation_click`
- `form_start`
- `form_submit`
- `lead`

А вот:

- `qualified_lead`
- `sale`

нужно считать именно офлайн-конверсиями, а не простыми фронтенд-событиями.
