# Настройка целей Matomo

Создайте 4 цели в Matomo (Admin -> Goals):

1. **WhatsApp Click**
- Type: Event
- Category: `MADCORE`
- Action: `whatsapp_click`

2. **Telegram Click**
- Type: Event
- Category: `MADCORE`
- Action: `telegram_click`

3. **Form Submit**
- Type: Event
- Category: `MADCORE`
- Action: `form_submit`

4. **Thanks Page View**
- Type: Visit a given page URL
- URL contains: `/thanks`
