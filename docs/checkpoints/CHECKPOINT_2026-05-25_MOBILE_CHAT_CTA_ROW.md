# CHECKPOINT 2026-05-25 MOBILE CHAT CTA ROW

## Что найдено

- На мобильной версии `MADCORE Gena` две кнопки открытых чатов в hero-блоке занимали две полные отдельные полосы и визуально были слишком крупными.
- Пользовательский запрос был точечный:
  - уменьшить только `Перейти в чат Telegram` и `Перейти в чат Max`;
  - поставить их в одну строку;
  - остальные CTA не менять.

## Что сделано

- В `src/app/globals.css` добавлено отдельное правило для мобильной версии `cta-grid--with-chat-links` на ширинах до `639px`.
- Кнопки `Перейти в чат Telegram` и `Перейти в чат Max` теперь:
  - стоят рядом в одной строке;
  - имеют уменьшенную высоту и внутренние отступы;
  - скрывают верхнюю подпись `Открытый чат` только на мобильной версии, чтобы остаться компактными.
- Остальные CTA в hero-блоке:
  - `Написать в WhatsApp`
  - `Написать в Telegram`
  - `Написать в MaX`
  - `Позвонить`
  остались без изменений по раскладке и размеру.

## Что проверено

- `npm run lint`
- `npm run build`

## Что осталось

- Если потребуется, эту же мобильную правку CTA можно отдельно выкатить на боевой `https://madcore.site`.
- Следующий отдельный незавершенный приоритет проекта по checkpoint не меняется:
  - прикладные воронки Яндекс.Метрики;
  - при необходимости `HeatmapSessionRecording` для Matomo.

## Следующая точка продолжения

1. Если продолжаем полировку hero-блока, сверять только мобильный вид CTA и не менять остальные кнопки без отдельного запроса.
2. Если правку нужно отправить на production, брать в выкладку минимум `src/app/globals.css` и затем повторять smoke-проверки.
